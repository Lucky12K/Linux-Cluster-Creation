# Creating a Linux Cluster for simulations with OpenFoam

¿Que es un Cluster?

Un cluster (o clúster de computación) es un conjunto de computadoras interconectadas que trabajan de manera coordinada como si fueran un solo sistema. Su objetivo principal es dividir y ejecutar tareas de forma paralela, permitiendo resolver problemas de mayor complejidad, reducir el tiempo de procesamiento y aumentar la confiabilidad del sistema en comparación con una sola máquina.

Cada computadora dentro del cluster, conocida como nodo, contribuye con sus propios recursos de cómputo —como procesador, memoria y almacenamiento— y se comunica con los demás nodos a través de una red. Gracias a esta arquitectura, los clusters son ampliamente utilizados en aplicaciones científicas, análisis de datos, simulaciones, servidores y entornos donde se requiere alto rendimiento o alta disponibilidad.
